Программа для поиска читов у игроков на пк. Была создана еще в 23 году. Так и не была отдана в руки админов сервера.

Установка:
1. Откройте проект, используя Visual Studio.
2. Переместите содержимое из папки "files for program" по пути C:/Program Files/Pro-Arena Checker/, иначе измените в коде на рабочую директорию или на нужнную. Я лично паковал программу, используя nuget пакет Costura.Fody, вы же можете использовать просто InstallForge для создания своего собственного Setup'a.
Удачи!
